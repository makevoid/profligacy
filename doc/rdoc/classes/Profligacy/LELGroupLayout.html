Class Profligacy::LELGroupLayout
In: lib/profligacy/lel.rb
Parent: Object

Used by Swing::Build as the listener for LELParser which configures a GroupLayout from the LEL expression. See ihate.rubyforge.org/profligacy/lel.html for instructions on how to use LEL.
